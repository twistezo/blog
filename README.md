# My programming blog

Live on https://twistezo.github.io/blog/

### Requirements

- Bundler

### Install packages

- `bundle install`

### Build and serve

- `bundle exec jekyll serve --livereload`
- `http://127.0.0.1:4000`

### Tools

- run

  `./tools/run.sh`

- tests

  `./tools/test.sh`
